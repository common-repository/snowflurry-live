Add an animated snow effect to your website based on live weather conditions anywhere in the world.

== Description ==

SnowFlurry Live - Sync with Current Snowfall is a unique WordPress plugin that brings the beauty of real-time snowfall to your website. This plugin uses live weather data from https://www.weatherapi.com/ (a weather API) to animate snow on your website, matching the current snow conditions from any location or locations around the globe.

Key Features:
- Live weather synchronization via https://www.weatherapi.com/: The snowfall on your site matches real-world conditions.
- Global coverage: Get accurate snowfall animations for any location worldwide.
- Easy configuration: Set up in seconds and customize as per your needs.

Coming Soon:
- More configuration options to customize the snowfall intensity and appearance.

== Installation ==

1. Upload the `snowflurry` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin by going to the SnowFlurry Live settings page in your admin panel.

== Frequently Asked Questions ==

= Where does the weather data come from? =

It comes from https://www.weatherapi.com/ and their API.

= Can I choose a specific location for the snowfall effect? =

Yes, SnowFlurry Live allows you to select any location or multiple locations in the world for the live snowfall effect.

= Is this plugin resource-intensive? =

No, SnowFlurry Live is designed to be lightweight and should not significantly impact your site's performance. However, it does disable itself on mobile devices and if the window is not focused.

== External Service Usage ==

This plugin makes use of WeatherAPI.com to fetch current weather conditions. Data is sent to WeatherAPI.com in order to retrieve weather information based on the configured settings.

* **WeatherAPI.com Service**: http://api.weatherapi.com/v1/current.json
* **Terms of Service**: https://www.weatherapi.com/terms.aspx
* **Privacy Policy**: https://www.weatherapi.com/privacy.aspx

Please review the terms and privacy policy of WeatherAPI.com to ensure compliance with their usage requirements.
